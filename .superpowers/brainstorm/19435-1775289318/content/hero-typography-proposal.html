Key Changes from Current

Before
  • Helvetica 1957 (not web-safe)
  • No labels/anchors
  • Intro text same size as hero
  • Inconsistent spacing
  • Hero text: font-weight 400
After
  • Satoshi (free, web-optimized)
  • Uppercase label above intro
  • Clear size hierarchy (H1 > Intro > Subtitle)
  • 8px spacing grid
  • Hero text: font-weight 700 (bolder)

What stays the same

All images, grid background, window system, animations (slide-up reveal), blur effects, logo flip animation, hover inline-image on "ultimate". Only typography sizing, font family, and hierarchy change.
